# UAS R&D Hub - Enhanced Version

輕量級、高效能無人機技術文件系統。

## 📖 網頁內容簡介

本網頁整合了無人機系統工程師與研發人員所需的核心知識，分為 **7 大主要章節**：

### 1. 專案概述 (Overview)
介紹 UAS R&D Hub 的設計理念。

### 2. 核心功能 (Key Features)
展示網頁提供的三大互動式功能：
- **單檔運作**：所有邏輯、樣式、內容封裝於一個 HTML 檔案
- **互動式視覺化**：Chart.js 圖表、動態 Pinout 介面、MathJax 公式渲染
- **使用者體驗優化**：全文檢索、書籤、進度條、深淺色主題切換、新手導覽

### 3. 技術棧 (Tech Stack)
列舉網頁採用的技術與工具：
- Semantic HTML5、Tailwind CSS、Chart.js、MathJax、FontAwesome、Vanilla JavaScript
- 所有資源透過 CDN 引入，無需編譯流程

### 4. 快速開始 (Quick Start)
提供四個簡易步驟快速上手：
1. 下載 HTML 檔案
2. 用瀏覽器開啟
3. 確保網路連接
4. 探索各項功能

### 5. 無損整合標準作業程序 (Integration SOP)
為開發者設計的四步驟指南，說明如何新增技術章節：
1. **判斷資料類別** - 分為 Strategic / Hardware / Logic / Soft/Ops
2. **建立內容區塊** - 提供 HTML 模板與 CSS 樣式
3. **更新導覽列** - 在側邊欄新增章節連結
4. **註冊搜尋索引** - 更新搜尋資料庫確保功能完整

### 6. 樣式指南 (Style Guide)
統一的設計規範：
- **背景與結構**：定義卡片、邊框的背景顏色
- **文字顏色**：標題、正文、輔助資訊的顏色層級
- **功能色系**：根據內容類型選擇對應顏色
  - 🟢 Emerald (綠) - 硬體規格、Datasheet
  - 🔵 Sky (藍) - 軟體架構、通訊協議
  - 🟠 Amber (橘) - 電源系統、警告資訊
  - 🔴 Rose (紅) - 核心處理器、安全機制
- **Badge 樣式**：資料來源分類標籤

### 7. 常見問題 (FAQ)
解答使用者與開發者的常見疑問：
- 搜尋功能不生效的排查
- 導航點擊無反應的解決方案
- 書籤功能使用說明
- 主題切換與離線使用方案
