實機通訊整合：RPi5 + Pixhawk 6C

本文件說明如何將機載電腦與飛控進行硬體連接，並透過 MAVSDK 實作手勢控制指令。

1. 硬體連線圖 (Hardware Connection)

使用 RPi5 的 UART0 與 Pixhawk 6C 的 TELEM2 進行交叉連線：

Pixhawk 6C (TELEM2)RPi 5 GPIO (Physical Pin)功能
TX (Pin 2)Pin 10 (RXD0 / GPIO 15)數據傳入 RPi
RX (Pin 3)Pin 8 (TXD0 / GPIO 14)指令傳入飛控
GND (Pin 6)Pin 6 (GND)共地
注意：RPi 5 建議由獨立 5V/5A 電源供電，切勿從飛控板取電。

2. 軟體環境 (Conda)

# 建立環境
conda create -n uas_vision python=3.10 -y
conda activate uas_vision

# 安裝通訊與視覺套件
pip install mavsdk mediapipe opencv-python
sudo usermod -a -G dialout $USER
